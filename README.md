# Beauty-Product-reviews-dataset
     Focuses on analyzing beauty product reviews using machine learning. 
     Synthetic dataset with 50,000 reviews, 200 products, and 1,000 users.    
     Includes sentiment, CTR, and time spent data.     
     Useful for recommender systems and NLP research.
